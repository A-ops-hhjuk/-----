# منصة صفحة - للدعم النفسي والاستشارات المهنية

## 🌟 نظرة عامة على المشروع

**صفحة** منصة احترافية متكاملة الواجهة الأمامية مخصصة لتقديم الدعم النفسي والاستشارات المهنية للعاملين في القطاع الإنساني. تم تصميمها بالكامل باستخدام HTML5 وCSS3 وJavaScript الخالص مع استخدام مكتبات CDN.

## ✅ الشاشات المكتملة

### 1. الصفحة الرئيسية (`index.html`)
- قسم Hero بتصميم متحرك مع بطاقة مستشار تفاعلية
- شارات الثقة (سرية - معتمدون - 24/7)
- قسم المميزات (6 مزايا)
- قسم الإحصاءات مع تأثير العداد المتحرك
- كيف تعمل المنصة (4 خطوات)
- عرض المستشارين (3 بطاقات)
- تبويبات الخدمات التفاعلية (4 تبويبات)
- آراء المستخدمين
- قسم حلول المنظمات مع معاينة لوحة التحكم
- أحدث المقالات
- قسم CTA للتسجيل

### 2. التسجيل وتسجيل الدخول (`auth.html`)
- واجهة تسجيل دخول بالبريد أو Google/LinkedIn
- نموذج تسجيل متعدد الخطوات (3 خطوات)
- اختيار نوع المستخدم (فرد / منظمة / مستشار)
- قياس قوة كلمة المرور
- خيار الدخول التجريبي (فردي / منظمة)
- الجانب المرئي مع المميزات

### 3. صفحة المستشارين (`consultants.html`)
- شريط بحث وفرز
- فلتر شامل (تخصص، لغة، سعر، تقييم، توافر)
- عرض 8 مستشارين في شبكة أو قائمة
- نافذة منبثقة لعرض الملف الكامل مع التقييمات
- ترقيم الصفحات
- بانر التوصية الذكية

### 4. حجز المواعيد (`booking.html`)
- مؤشر الخطوات (4 خطوات)
- معلومات المستشار المختار
- اختيار نوع الجلسة (مرئية/صوتية/نصية)
- اختيار المدة (25/50/80 دقيقة)
- تقويم تفاعلي بالكامل بـ JavaScript
- اختيار الوقت من فتحات متاحة/غير متاحة
- ملاحظات للمستشار
- نموذج دفع (بطاقة/Apple Pay/مدى/تحويل)
- ملخص الحجز جانبي ثابت
- نافذة نجاح التأكيد

### 5. التقييم النفسي الذكي (`assessment.html`)
- شاشة مقدمة مع إحصاءات وضمانات الخصوصية
- 20 سؤالاً موزعة على 4 محاور
- شريط تقدم متحرك
- أزرار اختيار بإيموجي تفاعلية
- تنقل بين الأسئلة (السابق/التالي + التقدم التلقائي)
- صفحة نتائج مفصلة مع أشرطة التقدم المتحركة
- توصيات مخصصة
- قائمة المستشارين الموصى بهم بنسبة التطابق

### 6. لوحة تحكم المستخدم (`dashboard-user.html`)
- navbar لوحة تحكم مخصص
- شريط جانبي مع قوائم وتحذير الطوارئ
- بانر ترحيبي مع مؤشر الرفاهية
- تتبع المزاج اليومي مع حقل ملاحظة
- 4 بطاقات إحصائية
- تفاصيل الجلسة القادمة مع عداد تنازلي
- مخطط المزاج الأسبوعي
- جدول سجل الجلسات
- نتائج آخر تقييم
- الموارد المقترحة

### 7. لوحة تحكم المنظمات (`dashboard-org.html`)
- header مؤسسي مع شارة "مجهول الهوية تماماً"
- 4 بطاقات إحصائية
- مخطط اتجاه الرفاهية (6 أشهر)
- مخطط توزيع المستويات بـ SVG
- شريط الاحتراق الوظيفي حسب الفريق
- توصيات الذكاء الاصطناعي (عاجل/مهم/إيجابي)
- إجراءات سريعة (دعوة - تقرير - ورشة)
- إحصاءات الاستخدام
- معلومات الاشتراك وباقة المؤسسات

### 8. المدونة (`blog.html`)
- بطاقة مقال مميز (hero article)
- فلاتر التصنيف (6 تصنيفات)
- شبكة 6 مقالات مع كارت احترافي
- شريط جانبي: بحث، تصنيفات، أكثر قراءة
- صندوق اشتراك النشرة البريدية
- CTA التقييم

### 9. من نحن (`about.html`)
- hero بخلفية متدرجة
- قسم الرسالة مع اقتباس ملهم
- قسم القيم الأساسية (4 قيم)
- بطاقات الفريق (4 أعضاء)
- خط زمني تفاعلي (5 مراحل)
- الاعتمادات والجوائز
- شعارات الشركاء

### 10. تواصل معنا (`contact.html`)
- بانر الطوارئ النفسية
- معلومات التواصل (بريد، هاتف، واتساب، طوارئ)
- اختيار موضوع التواصل (6 أنواع)
- نموذج تواصل كامل مع التحقق
- قسم الأسئلة الشائعة (FAQ) أكوردون تفاعلي
